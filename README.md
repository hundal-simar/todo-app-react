# 📝 Todo Apps (React)

This repository contains two versions of a Todo application built using React:

---

## 📌 Projects Included

### 1. Context API Todo App

* Uses React Context API for state management
* Simple and lightweight approach

📂 Folder: `context-api-todo`

---

### 2. Redux Todo App

* Uses Redux Toolkit for state management
* Scalable and structured approach

📂 Folder: `redux-toolkit-todo`

---



## 🚀 Purpose

This project was built to understand and compare different state management approaches in React.

---

## 🛠️ Tech Stack

* React
* JavaScript
* Tailwind CSS
